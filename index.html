README

CTN 治験届システム化 プロトタイプ

ブラウザでそのまま動作・操作できる共有用プレビュー（アカウント不要）

社内共有用の試作です。URL を知る人は誰でも閲覧できます（検索エンジンには noindex 済み）。

共有ドキュメント

  • UX モック (React) — 治験届管理のメイン画面。操作可能なインタラクティブ版
  • 操作マニュアル — UX モックの操作ガイド
  • システム化 検討まとめ — データモデル・順序番号・外字の設計判断と CSV/GxP 要点
  • XSD マッピング表（たたき台） — 手引き第4版から自動抽出した 138 要素 × STATUS × 順序番号型
  • 届出種別 × 項目 使用マトリクス（たたき台） — 計画/変更/終了/中止/開発中止で各項目をどう扱うか（記入/継承/3.8固定NONE）
  • 静的バンドル一式 — LP・ドキュメントビューア等を含む生成済みサイト
