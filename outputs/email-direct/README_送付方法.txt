CTN UXモック 送付方法

このフォルダの CTN_UX_Mock_DIRECT.html をメールに添付して送付してください。

メール本文では、以下を必ず案内してください。

- 添付HTMLをPCに保存する
- Microsoft Edge または Google Chrome で開く
- Outlook、Teams、SharePoint上のプレビューでは開かない

このHTMLは単一ファイルです。CSSとJavaScriptを内部に埋め込んでいるため、別ファイルやサーバーは不要です。

注意:
社内メールのセキュリティ設定によっては、HTML添付がブロックされる場合があります。
その場合は、Azure Static Web AppsなどのURL公開方式に切り替える必要があります。
