CTN共有パッケージの開き方

1. ZIPを右クリックして「すべて展開」してください。
2. 展開されたフォルダの index.html または OPEN_ME_FIRST.html を開いてください。
3. ZIPの中から直接HTMLを開くと、ブラウザやメールソフトの制限で真っ白になることがあります。
4. それでも表示されない場合は、Microsoft Edge または Chrome で開いてください。

主なファイル:
- index.html
- ctn-ui.html
- ctn-systemization-summary.html
- ctn-system-lp.html
- pharma-translate-lp.html
- docs-html/index.html
- ppt/ctn-workflow-improvement-summary-v4.pptx
