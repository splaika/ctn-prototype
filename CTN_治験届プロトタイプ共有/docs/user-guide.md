# ユーザーガイド

## 1. ダッシュボードを見る

1. 左メニューで `ダッシュボード` を開く。
2. 製品別ステータスで、初回届、変更届回数、進行中ドラフト、次回期限を見る。
3. 期限管理・通知予定で、6か月/12か月期限が近い変更を確認する。
4. レビュー中、承認待ち、期限注意の届出を優先して処理する。

## 2. 既存PDFを取り込む

1. `既存PDF取込` を開く。
2. PDFをアップロードする。
3. 取込ジョブの状態を確認する。
4. 抽出候補を確認する。
5. 誤認識や未抽出項目を修正する。
6. 確認完了後、確定版DB化する。

## 3. 変更届を作る

1. ダッシュボードまたは取込済み版から対象製品を選ぶ。
2. 最新の確定版から新しい変更届ドラフトを作成する。
3. 届出概要で届出種別、届出区分、届出回数を確認する。
4. 必要な入力メニューで変更内容を入力する。
5. 医師異動は `医師情報` で追加、削除、氏名変更をイベント行として入力する。
6. 治験使用薬は順序番号を確認し、詰め直さない。

## 4. 期限リマインダーを確認する

1. ダッシュボードの `期限管理・通知予定` を見る。
2. 実変更日、期限日、残日数を確認する。
3. 担当者は期限前に変更届ドラフトを起票する。
4. 承認者は通知を受けたらレビュー/承認を進める。

## 5. 外字を確認する

1. `外字・表記` を開く。
2. 原表記と届出用表記を確認する。
3. 置換が妥当であれば確認済みにする。
4. 不適切な場合は届出用表記を修正する。
5. 未確認の外字が残っている場合、正式出力には進めない。

## 6. 承認して出力する

1. ダッシュボードで対象届出を確認する。
2. レビュー担当が入力、外字、添付資料、順序番号を確認する。
3. 承認者が電子署名付きで承認する。
4. 承認後にPDFイメージを確認する。
5. 問題なければXML/PDFを出力する。
6. 出力物はSharePoint `Generated Outputs` に保存される。
